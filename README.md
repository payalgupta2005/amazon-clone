# Amazon Clone

A responsive Amazon landing page clone built using HTML and CSS. This project focuses on recreating the user interface of Amazon's homepage while practicing modern web design and responsive layouts.

## Features

* Responsive homepage design
* Navigation bar and search section
* Product categories and cards
* Hero banner section
* Footer layout similar to Amazon
* Clean and structured UI

## Tech Stack

* HTML5
* CSS3
